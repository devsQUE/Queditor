About Queditor

Queditor is built for Android developers who want a powerful, clean, and reliable code editing experience. With modular design, file management, and live preview, it’s your go-to tool for coding on the go.

Features

Create & manage projects

Organize multiple projects inside the app with a clean, modular file tree.

File Management

Add, rename, delete, and organize files & folders with ease.

Import ZIP as project

Bring an existing project in one step. Folders and files are reconstructed automatically.

Export project as ZIP

Share or back up your work instantly with a single ZIP export.

Auto-Save

Your code is always safe with built-in auto-save powered by IndexedDB.

Syntax highlighting

Readable code with language‑aware highlighting for web stacks and more.

Auto completion

Smarter editing with suggestions as you type.

Code formatting

Keep your code tidy and consistent with built‑in formatting.

Code Preview

Instant preview of your projects (Only index.html)

Undo / Redo

Fearless editing with multi‑step undo and redo controls.

Search & replace

Locate and rewrite across the open file quickly.

Copy / Move via drag & drop

Drag and drop to copy or move files/folders where you want them.
